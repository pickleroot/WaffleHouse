# Better College Schedule

A faster, cleaner browser app for viewing your college timetable and searching courses/teachers/rooms.

Works better than the official college website.

## Why people use it

- Loads much faster
- Actually usable search
- Easy to read weekly view

## Main features

- Quick search for courses, teachers, rooms
- Weekly timetable view
- Day list view
- Dark / light mode
- Works offline (uses last loaded data)
- Mobile friendly

## How to use

1. Open https://yourusername.github.io/college-better-schedule/  
   (or wherever it's hosted)

2. Use it

# Better College Schedule

A faster, cleaner browser app for viewing your college timetable and searching courses/teachers/times.

Works better than the official college website.

## Getting started

```sh
# Clone repo
git clone https://github.com/TS-24/WaffleHouseV1.git
cd WaffleHouseV1

# Frontend
cd frontend
npm run dev  # or pnpm run dev
```
